## API

| 属性     | 说明                    | 类型             | 默认值 |
| -------- | ----------------------- | ---------------- | ------ |
| checked  | 指定当前是否选中        | boolean          | false  |
| disabled | 是否禁用                | boolean          | false  |
| small    | 是否为小型 Switch       | boolean          | false  |
| onChange | Switch 变化时的回调函数 | (e: any) => void | -      |
