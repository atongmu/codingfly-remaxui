## API

| 属性      | 说明         | 类型          | 默认值 |
| --------- | ------------ | ------------- | ------ |
| title     | 标题         | string        | -      |
| padding   | 内边距       | boolean       | false  |
| lightBg   | 浅色背景     | boolean       | false  |
| grayBg    | 灰色背景     | boolean       | false  |
| style     | 设置按钮样式 | CSSProperties | -      |
| className | 包裹层的类名 | string        | -      |
