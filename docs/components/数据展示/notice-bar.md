## API

#### NoticeBar

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| icon | 图标 | string | `0` |
| color | 文字颜色 | string | - |
| backgroundColor | 背景颜色 | string | - |
| data | 字符串数组  | Array<string> | - |
| autoplayInterval | 自动切换的时间间隔  | number | 3000 |
