## API

| 属性      | 说明             | 类型       | 默认值 |
| --------- | ---------------- | ---------- | ------ |
| show      | 是否展示遮罩层   | boolean    | false  |
| zIndex    | 层级             | number     | 9999   |
| className | 包裹层的类名     | string     | -      |
| style     | 设置样式         | string     | -      |
| onTap     | 点击按钮时的回调 | () => void | -      |
